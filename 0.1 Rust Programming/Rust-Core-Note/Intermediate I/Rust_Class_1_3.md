
# Type Language

Dynamic <=> Static

Dynamic = Data Type을 순전히 Complier가 결정
Static -> Data Type을 순전히 Complier가 결정 + 개발자가 보조

# Mission Critical Language

사용자 <-> 스마트폰 <-> 웹서버

사용자 <-> 스마트폰 Front end Language: JS/TS
스마트폰 <-> 웹서버 Back end: 
Hight level: Pyhton, Ruby, Elixir
Lower level: C, C++, Rust
