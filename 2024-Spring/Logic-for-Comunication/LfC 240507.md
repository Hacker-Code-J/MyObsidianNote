# # Direct

- $J\implies E,\lnot J\quad/\quad\lnot E$

| J   | $\supset$ | E,  | $\sim$ J | /$\sim$E |
| --- | --------- | --- | -------- | -------- |
| T   | T         | T   | F        | F        |
| T   | F         | F   | F        | T        |
| F   | T         | T   | T        | F        |
| F   | T         | F   | T        | T        |

-> It is invalid

- $A\supset\sim R,\ \sim R\supset K\quad / \quad A\supset K$
-> It is valid
