# #8 Material Equivalence (Equiv)

- $(p\equiv q)$ :: $[(p\supset q)\cdot (q\supset p)]$

- $[(p\cdot q)\supset r]$ :: $[p\supset (q\supset r)]$

- $p$ :: $(p\lor p)$
- $p$ :: $(p\cdot p)$
